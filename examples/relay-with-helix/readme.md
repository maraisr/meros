# Relay + Helix

Example demonstrating how `meros` can be used to stream defer'd GraphQL patches with relay.

## Setup

1. Install

```sh
pnpm i
```

2. Start

```sh
pnpm run start
```

3. Load up:

- _localhost:3004_ is our GraphQL Helix server
- _localhost:8080_ is where our Relay/React SPA is running
